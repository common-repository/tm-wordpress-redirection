== Description ==

Redirect link from another web with rel = nofollow like vBulletin or normal redirect.

Just activate and settings.....

== Installation ==

1. Upload `tm auto post` folder to the `/wp-content/plugins/` directory .
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Settings can be configured in the 'Options' area of your blog.
